# 🐾 Animal Park / Pet Shop Web Application

A full-stack web application designed to provide users with an interactive platform to browse, adopt, or purchase pets online. The system offers a seamless user experience with authentication, dynamic data handling, and an admin panel for management.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 🐶 Browse Pets / Products
- 🔍 Search and Filter Functionality
- 🛒 Add to Cart / Adoption Requests
- 🧑‍💼 Admin Dashboard (Manage Users, Pets, Orders)
- 📱 Responsive Design (Mobile + Desktop)
- ⚡ Real-time Data Handling
